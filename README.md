# ⚡ Household Energy Consumption Analytics

Live App: [Click Here](https://energy-analytics-dashboard-lldet9recvaafzqo95irm8.streamlit.app/)

## What it does
- Analyses 1.4M+ real household power consumption records
- Detects peak usage hours and consumption patterns
- Predicts energy demand using Random Forest (R² = 0.9990)
- Interactive dashboard with live sliders for prediction

## Tech Stack
Python | Pandas | Scikit-learn | Streamlit | Matplotlib | Seaborn

## Dataset
UCI Household Electric Power Consumption (2006–2010)
